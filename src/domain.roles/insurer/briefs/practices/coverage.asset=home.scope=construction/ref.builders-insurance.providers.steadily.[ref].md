## who can apply

From [Steadily Builder's Risk Guide](https://www.steadily.com/blog/ultimate-guide-to-builders-risk-insurance):
> "Builder's risk insurance is a crucial tool for landlords renovating or constructing rental properties."

**target use cases:**
- fix and rent projects
- major remodels
- new construction for rental properties

**NOT for:**
- owner-occupied homes (use other carriers)
- contractor-purchased policies
- commercial development

---

## acquisition process

### step 1: gather project details

| required info | notes |
|---------------|-------|
| property address | must match permit |
| project type | renovation vs new construction |
| project value | total cost (materials + labor) |
| project timeline | start date + expected completion |
| completion % | must be <30% at inception |

### step 2: request quote

- online at steadily.com
- quotes generated "in minutes"
- integrates with 400+ property tech platforms for faster quotes

### step 3: carrier assignment

you do NOT choose the carrier — Steadily assigns based on:
- property location
- project type
- risk profile

carriers include:
- Obsidian Insurance (A- rated)
- Main Street America
- Canopius
- Markel
- Fortegra
- Steadily Insurance Company (own carrier since Q4 2024)

### step 4: bind coverage

- policy documents issued
- coverage begins on effective date

## claims process

From [Coverage Cat](https://www.coveragecat.com/reviews/steadily-insurance-review):
> "You can file a claim via the Steadily app or website, but Steadily does not adjust the claim. They pass the information to the underlying carrier (e.g., Canopius or Markel), who will assign an adjuster."

**key points:**
- file via app or website
- Steadily passes to assigned carrier
- carrier assigns adjuster
- Snapsheet partnership (Feb 2024) for digital claims

**caution:** extra layer between you and adjuster may cause delay.
