## 🔑 licenser

- **scale**: credential-level, professional authorization
- **focus**: license verification, status lookup, registry access
- **maximizes**: trust in contractor credentials via authoritative sources

used to verify professional licenses (contractors, electricians, plumbers) are legitimate and unexpired.

### why the key?

a license is a key — it unlocks the right to practice:

- driver's license = key to drive
- contractor license = key to build
- medical license = key to practice medicine

the 🔑 licenser verifies who holds the keys.
