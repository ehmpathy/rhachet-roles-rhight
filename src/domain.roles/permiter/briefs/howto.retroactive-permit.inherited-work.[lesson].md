# howto.retroactive-permit.inherited-work

## .what

retroactive permit process for work done by a PREVIOUS OWNER that you inherited when you bought the property.

## .why

- you didn't do the work — you can't vouch for quality
- city inspector needs someone qualified to certify it
- licensed contractor substitutes for the unknown original installer
- municipalities often show leniency for innocent purchasers
- higher cost but eliminates insurance denial risk

---

## use case

you:
- bought a house with unpermitted work (electrical, plumbing, HVAC, additions)
- previous owner did the work (or hired someone who didn't permit it)
- you discovered it via permit search, equipment age mismatch, or sale inspection
- now need to legitimize for insurance protection, sale, or refinance

---

## why this path differs from own-work

| factor | own work | inherited work |
|--------|----------|----------------|
| who did the work | you | unknown |
| who certifies quality | city inspector | licensed contractor |
| can you answer inspector questions | yes | no |
| contractor certification required | no | yes |
| cost | lower | higher |
| leniency available | sometimes | often (innocent purchaser) |

**key difference**: you can't testify to how the work was done. someone qualified must assess it.

---

## the process

### step 1: document the situation

| document | purpose |
|----------|---------|
| purchase date | proves you didn't do the work |
| home inspection report | may document prior work |
| permit search results | shows gap in permit history |
| equipment photos | manufacture dates, installation quality |

### step 2: hire licensed contractor

for each trade (electrical, plumbing, HVAC):

| task | purpose |
|------|---------|
| inspect current work | assess code compliance |
| identify deficiencies | list what needs correction |
| provide written assessment | documentation for permit office |
| certify compliance (if compliant) | affidavit substitutes for original inspection |
| pull retroactive permit | submit as-built application |

**contractor requirements by trade**:

| trade | who can certify |
|-------|-----------------|
| electrical | master electrician |
| plumbing | master plumber |
| HVAC | licensed HVAC contractor |
| structural | licensed GC or engineer |

### step 3: apply for retroactive permit

| element | notes |
|---------|-------|
| contractor submits application | as "as-built" or "retroactive" |
| include contractor assessment | required for approval |
| explain previous owner did work | triggers innocent purchaser leniency |
| provide purchase records | proves timeline |

### step 4: request leniency

innocent purchaser accommodations to request:

| accommodation | how to request |
|---------------|----------------|
| penalty waiver | "I bought this property with unpermitted work" |
| deadline flexibility | "I need time to make corrections" |
| code version flexibility | "Work was done before I purchased" |

> "If you are an innocent purchaser, the building department will likely work with you, possibly offer deadline flexibility and waive penalties."
> — [Quora: Previous owner unpermitted work](https://www.quora.com/If-a-previous-home-owner-had-additional-construction-without-permit-to-the-house-ex-additional-bathroom-and-other-type-of-construction-that-required-city-permit-does-the-next-owner-will-inherit-or-responsible-for)

### step 5: city inspection

| what happens | notes |
|--------------|-------|
| inspector reviews work | contractor present to answer questions |
| contractor certification accepted | substitutes for your testimony |
| corrections identified | if any code issues found |

### step 6: corrections (if needed)

| scenario | action |
|----------|--------|
| minor deficiencies | contractor fixes |
| major deficiencies | scope work, fix, re-inspect |
| passed | receive final approval |

### step 7: final approval

| outcome | benefit |
|---------|---------|
| permit finalized | legal record created |
| property records updated | searchable by insurers |
| insurance denial risk eliminated | permit proves inspection |
| resale value protected | no unpermitted work disclosure needed |

## cost breakdown (typical)

| item | range | notes |
|------|-------|-------|
| contractor assessment | $150-400 | per trade |
| permit fee | $100-400 | varies by work type |
| retroactive penalty | 0-2x permit fee | often waived for innocent purchaser |
| corrections (if needed) | varies | contractor labor + materials |
| re-inspection | $50-100 each | if fail first inspection |
| **total per system** | **$400-1,200** | higher than own-work path |

**cost comparison**:

| path | typical total |
|------|---------------|
| own work | $150-1,000 |
| inherited work | $400-1,200 |
| difference | contractor certification fee |

## leniency documentation

municipalities that offer innocent purchaser leniency:

| jurisdiction | documented policy |
|--------------|-------------------|
| Indianapolis, IN | "DBNS typically works with innocent purchasers" |
| general | "building department will likely work with you" |
| some areas | grace period to bring into compliance |

**how to trigger**:
- explain upfront that you inherited the work
- show purchase records
- voluntary disclosure (not caught)
- cooperative attitude

---

## risks

| risk | mitigation |
|------|------------|
| work is substandard | contractor identifies before you pay permit fees |
| corrections expensive | get contractor estimate first |
| can't find who did work | contractor certification substitutes |
| original contractor out of business | contractor certification substitutes |

---

## when to use this path

| scenario | recommendation |
|----------|----------------|
| previous owner did the work | YES — this path (required) |
| you don't know who did it | YES — this path |
| work was done before purchase | YES — this path |
| you did the work yourself | NO — use own-work path |
