# ref.homeowner-permit-exemption.indianapolis-in

## .what

homeowners in Indianapolis/Marion County can pull their own electrical permits for work on their primary residence without a contractor license.

## .why this matters

- homeowners may perform their own electrical work legally
- no contractor license required for owner-occupied residential property
- work must still meet code and pass inspections
- this is a local policy, not state law

## .the rule

per Indianapolis DBNS policy:

> "Any construction requiring a permit requires the applicant to be either a Marion County licensed contractor or the property owner."

and:

> "In most cases, homeowners in Indianapolis can pull permits and perform work on their own primary residence as an owner-builder."

## .homeowner responsibilities

even without a contractor license, homeowners must:

| requirement | detail |
|-------------|--------|
| meet code | all work must comply with adopted electrical code |
| pass inspections | DBNS inspector verifies work before closeout |
| provide documentation | load calculations, panel schedules as required |
| primary residence | exemption applies to owner-occupied property only |

## .what homeowners cannot do

| restriction | reason |
|-------------|--------|
| work on rental property | owner-builder exemption requires occupancy |
| work for others | that requires contractor registration |
| skip inspections | inspections required regardless of who does work |
