# ref.permit-application-requirements.indianapolis-in

## .what

documentation required to apply for an electrical permit in Indianapolis/Marion County.

## .why this matters

- applicants need correct documents before submission
- incomplete applications delay permit issuance
- requirements vary by project complexity

## .permit authority

| field | value |
|-------|-------|
| department | Department of Business and Neighborhood Services (DBNS) |
| address | 1200 Madison Ave, Suite 100, Indianapolis, IN 46225 |
| phone | (317) 327-8700 |

## .required documents

### all electrical permits

| document | description |
|----------|-------------|
| permit application | property address, owner info, work description |
| applicant identification | homeowner proof or contractor license number |

### panel upgrades / service changes

| document | description |
|----------|-------------|
| electrical load calculation | per NEC Article 220 |
| panel schedule | circuits, amperage, wire gauge, purpose |

### complex projects (new circuits, rewire, additions)

| document | description |
|----------|-------------|
| electrical plan | outlet/switch locations, circuit assignments |
| GFCI/AFCI locations | per NEC requirements for wet/arc-fault areas |

### contractor applications

| document | description |
|----------|-------------|
| contractor license number | Marion County registration |
| proof of insurance | liability coverage |

## .load calculation basics (NEC Article 220)

for panel upgrades, DBNS requires load calculation to verify panel size. basic formula:

```
general load = (floor area × 3 VA/sq ft)
small appliance circuits = 1500 VA × 2
laundry circuit = 1500 VA
fixed appliances = nameplate VA
largest motor = 125% of nameplate
```

## .inspection requirements

after permit issuance and work completion:

| inspection | when required |
|------------|---------------|
| rough-in | before walls closed |
| final | after work complete, before energize |
