# howto.shore-up-inherited-unpermitted-work

## .what

practical guide to legitimize inherited unpermitted work via retroactive permits — eliminate insurance denial risk for ~$500-1,500 per system.

## .why

- insurance can deny claims if unpermitted work caused damage (see `hazard.prepurchase-unpermitted-work.summary.tldr.md`)
- 10+ years of reliable operation proves function, NOT code compliance
- retroactive permits are straightforward and relatively inexpensive
- municipalities often show leniency for innocent purchasers
- once permitted, insurance denial risk is eliminated

## the process

### step 1: identify the work

document what was done without permits:

| method | what it reveals |
|--------|-----------------|
| permit search | what's on record vs. what exists |
| equipment age | manufacture dates vs. house age |
| material inspection | modern materials in old house |
| home inspection report | may note visible upgrades |

### step 2: hire licensed contractor

for each trade (electrical, plumbing, HVAC):

| task | purpose |
|------|---------|
| inspect current work | assess code compliance |
| certify compliance | provide affidavit if compliant |
| identify deficiencies | list corrections needed |
| pull retroactive permit | submit "as-built" application |

**key point**: the contractor certifies the work, substitutes for original inspection.

### step 3: apply for retroactive permit

| application element | notes |
|---------------------|-------|
| mark as "as-built" or "retroactive" | standard designation |
| include contractor assessment | required for approval |
| disclose previous owner did work | triggers leniency policies |
| provide photos of work | supports application |

### step 4: city inspection

inspector examines:

| item | what they check |
|------|-----------------|
| installation quality | proper methods used |
| code compliance | meets applicable code |
| safety | no hazards present |
| access | adequate clearances |

### step 5: corrections (if needed)

if work fails inspection:

| scenario | action |
|----------|--------|
| minor deficiency | contractor fixes, re-inspect |
| major deficiency | scope work, fix, re-inspect |
| cannot be made compliant | rarely happens for functional work |

### step 6: final approval

once passed:

| outcome | benefit |
|---------|---------|
| permit finalized | legal record created |
| property records updated | searchable by insurers |
| certificate of compliance | proof for insurance/sale |
| insurance risk eliminated | denial pathway closed |

## municipality leniency for inherited work

### what jurisdictions typically offer

| accommodation | description |
|---------------|-------------|
| penalty waiver | standard fee instead of 2x-3x |
| deadline flexibility | extended timeline for corrections |
| code version flexibility | may allow original code vs. current |
| grace period | time to bring into compliance |

### how to trigger leniency

| action | effect |
|--------|--------|
| explain you're innocent purchaser | activates leniency policies |
| show purchase records | proves when you acquired |
| voluntary disclosure | better treatment than if caught |
| cooperative attitude | inspectors have discretion |

## amnesty programs

some jurisdictions offer periodic amnesty with reduced fees and relaxed requirements.

### california statewide (AB 2533)

effective January 1, 2025:

| benefit | description |
|---------|-------------|
| 5-year grace period | extended timeline to bring ADUs to code |
| no impact fees | waived for legalization |
| code flexibility | cannot deny permit for code violations |

source: [Santa Cruz County: AB 2533](https://cdi.santacruzcountyca.gov/UPC/BuildingPermitsSafety/BuildingPermitsIndex/StateLaws/AB2533.aspx)

| focus | unpermitted dwell units |
| benefit | inspection without penalties |

source: [Berkeley Amnesty Program](https://berkeleyca.gov/construction-development/permits-design-parameters/permit-process/amnesty-program-unpermitted)

| feature | detail |
|---------|--------|

| benefit | legalize without penalties |
| focus | health and safety compliance |

source: [Riverside Safety-First Amnesty](https://riversideca.gov/business/building-safety/safety-first-amnesty-program)

### new jersey (hillsborough township)

| feature | detail |
|---------|--------|
| duration | 30 days (ended April 25, 2025) |

## "10+ years reliable" fallacy

| assumption | reality |
|------------|---------|
| "it's worked for 10 years" | proves function, not code compliance |
| "no problems so far" | problems happen suddenly (fire, flood) |
| "grandfathered in" | unpermitted work is not grandfathered |
| "they'd never check" | adjusters search permit records at claim time |

**why it doesn't protect you**:

insurers don't care about track record. they care about:
1. was it inspected?
2. is there a permit on record?
3. did unpermitted work cause the damage?

> "That you're 'grandfathered-in' is naive and probably costly to resolve."
> — [Garage Journal Forum](https://www.garagejournal.com/forum/threads/unpermitted-work-it-was-there-when-i-bought-the-place.498352/)

## prioritization strategy

if multiple systems need permits, prioritize by:

| priority | work type | reason |
|----------|-----------|--------|
| 1 | electrical | fire = catastrophic claim |
| 2 | HVAC/gas | fire/CO = catastrophic claim |
| 3 | plumbing | flood = expensive claim |
| 4 | structural | collapse = catastrophic |
| 5 | cosmetic additions | lower claim risk |

---

## when NOT to pursue retroactive permit

| scenario | recommendation |
|----------|----------------|
| work cannot be inspected (sealed in walls) | may require open walls |
| work is clearly substandard | fix before permit |
| sale/demolition planned soon | may not be worth cost |
| amnesty program scheduled | wait for better terms |

---

## checklist: shore up inherited unpermitted work

- [ ] pull permit history for property
- [ ] identify gaps (modern equipment without permits)
- [ ] prioritize by claim risk (electrical > plumbing > cosmetic)
- [ ] check for amnesty programs in your jurisdiction
- [ ] hire licensed contractor for each trade
- [ ] contractor inspects and certifies current work
- [ ] apply for retroactive permit (mark "as-built")
- [ ] explain you're innocent purchaser (triggers leniency)
- [ ] complete city inspection
- [ ] make corrections if needed
- [ ] obtain final approval and certificate
- [ ] keep documentation for insurance and future sale
