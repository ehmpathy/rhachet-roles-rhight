# howto.retroactive-permit.own-work

## .what

retroactive permit process for work YOU did on your own primary residence without a permit.

## .why

- homeowners can often pull their own permits (owner-builder exemption)
- you know the work quality because you did it
- city inspector verifies code compliance directly — no contractor certification needed
- lower cost path to legalization

---

## use case

you:
- did electrical/plumbing/HVAC/structural work yourself
- skipped the permit (didn't know, forgot, avoided hassle)
- now need to legitimize for sale, refinance, or insurance

---

## who can use this path

| requirement | detail |
|-------------|--------|
| owner-occupied | your primary residence |
| you did the work | you can answer inspector questions |
| work is accessible | inspector can see it (or you open walls) |
| jurisdiction allows | most do for owner-builders |

**check your jurisdiction**: some require licensed contractors for certain trades (electrical, plumbing, gas) even on owner-occupied property.

---

## the process

### step 1: contact permit office

| action | purpose |
|--------|---------|
| call or visit | explain you did work without permit |
| ask about owner-builder retroactive permits | confirm you qualify |
| ask about penalties | some waive for voluntary disclosure |
| get application requirements | what docs you need |

**tip**: voluntary disclosure typically gets better treatment than if they discover it.

### step 2: prepare documentation

| document | purpose |
|----------|---------|
| photos of work | show what was done |
| materials receipts | prove quality of materials |
| as-built drawings | show scope (may need to sketch) |
| load calculations (electrical) | if panel work involved |

**note**: some jurisdictions require architect/engineer drawings even for owner work. ask first.

### step 3: submit application

| element | notes |
|---------|-------|
| mark as "as-built" or "retroactive" | standard designation |
| pay permit fee | may include penalty (often 2x) |
| schedule inspection | typically 1-2 weeks out |

### step 4: inspection

inspector examines your work directly:

| what they check | your role |
|-----------------|-----------|
| code compliance | answer questions about how you did it |
| safety | show them the work |
| quality | explain your methods |
| access | provide ladder, move items, etc. |

**key advantage**: you can explain your work. you know where wires run, how you grounded, what materials you used.

### step 5: corrections (if needed)

| scenario | action |
|----------|--------|
| minor issues | fix yourself, schedule re-inspection |
| major issues | fix yourself or hire contractor |
| passed | receive final approval |

### step 6: final approval

| outcome | benefit |
|---------|---------|
| permit closed | legal record exists |
| property records updated | searchable by insurers, buyers |
| insurance risk reduced | permit proves inspection |

## cost breakdown (typical)

| item | range | notes |
|------|-------|-------|
| permit fee | $100-400 | varies by work type and jurisdiction |
| retroactive penalty | 1-2x permit fee | some waive for voluntary disclosure |
| re-inspection fee | $50-100 each | if you fail first inspection |
| documentation | $0-500 | if drawings required |
| **total** | **$150-1,000** | lower end of retroactive permit costs |

**key savings**: no contractor certification fee ($150-400) since you did the work and can answer to inspector directly.

---

## timeline

| phase | duration |
|-------|----------|
| application review | 1-2 weeks |
| inspection schedule | 1-2 weeks |
| corrections (if needed) | varies |
| re-inspection | 1-2 weeks |
| **total** | **3-6 weeks typical** |

## risks

| risk | mitigation |
|------|------------|
| inspector finds code violations | be prepared to fix |
| work is substandard | fix before you apply |
| need to open walls | factor in repair costs |
| jurisdiction requires contractor | check before you start |

---

## when to use this path

| scenario | recommendation |
|----------|----------------|
| you did quality work yourself | YES — this path |
| you know the work details | YES — this path |
| work is straightforward | YES — this path |
| you're unsure of quality | hire contractor to assess first |
| work is complex/structural | may need engineer regardless |

## indianapolis specifics

per `ref.homeowner-permit-exemption.indianapolis-in`:

> "Any construction that requires a permit requires the applicant to be either a Marion County licensed contractor or the property owner."

Indianapolis allows owner-builders to pull their own retroactive permits for primary residence work.

| requirement | indianapolis rule |
|-------------|-------------------|
| primary residence | yes, required |
| contractor license | not required for owner |
| inspection | required |
| code compliance | must meet adopted code |
