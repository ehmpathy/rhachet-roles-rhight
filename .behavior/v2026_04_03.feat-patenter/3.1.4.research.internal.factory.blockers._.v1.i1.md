
## access and credentials

### required credentials

| credential | required for | status | who provisions |
|------------|--------------|--------|----------------|
| USPTO ODP API key | patent.priors.search, patent.priors.fetch | **needed** | self-service registration |
| PatentsView API key | analytics queries (future) | **optional for v1** | self-service registration |

### registration process

**USPTO ODP**:
1. visit data.uspto.gov
2. create account
3. generate API key
4. store in keyrack or environment

**PatentsView**:
1. visit patentsview.org/apis
2. request API key
3. store in environment

### blocked without credentials

- **patent.priors.search**: cannot query USPTO API
- **patent.priors.fetch**: cannot retrieve patent documents

### workaround for development

use curl with test API key in environment. do not commit credentials.

```bash
# in ~/.config/rhachet/apikeys.env or similar
USPTO_API_KEY=your-key-here
PATENTSVIEW_API_KEY=your-key-here
```
