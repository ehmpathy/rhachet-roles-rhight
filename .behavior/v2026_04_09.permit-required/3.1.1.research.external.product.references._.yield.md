## .proof chain foundation

These references establish the complete proof chain for permit-required determination:

```
1. IRC R105.1 (baseline)     →  permits required for MEP work
        ↓
2. IRC R105.2 (exemptions)   →  enumerated exempt work
        ↓
3. State adoption docs       →  which IRC version applies
        ↓
4. Local ordinances          →  enforcement mechanism
        ↓
5. Case law (Veeck, Georgia) →  codes are public domain when adopted
        ↓
6. Trade standards (NECA)    →  workmanship requirements
        ↓
7. Penalty structures        →  consequences of non-compliance
```

## .usage

These references support:
- Citing exact code text in permit determinations
- Establishing jurisdiction hierarchy (federal → state → local)
- Proving public access to adopted codes
- Documenting enforcement consequences
- Grounding academic credibility of the system
