# blueprint: factory

## summary

**one factory build required**: Playwright-based scrape module for permit portal access.

the scrape module enables browser automation to search and fetch permits from the Indianapolis Accela portal. without this, permit.search and permit.fetch cannot be built.

extant factory infra (test-fns, jest, cache patterns, treestruct output) is reusable from patenter role.

## build sequence

1. install Playwright devDependency
2. create `src/utils/scrape/` module structure
3. implement launchBrowser, fillForm, extractTable utilities
4. create Indianapolis Accela selector file
5. add unit tests for utilities
6. capture HTML fixtures from live portal
7. add integration test for selector validity
8. verify factory readiness checklist

once factory is ready, product build can proceed.
