## subcomponent contracts

### permit.check.required (thought route)

```
given('permit.check.required route contract')
  then('instantiates route at .route/v{date}.permit.check.required/')
  then('creates 0.query.md pre-filled with postal and work')
  then('creates all research stones: work-interpretation, code.baseline, code.exemptions, jurisdiction.state, jurisdiction.local')
  then('creates distill stone: proof-chain')
  then('creates blueprint stone: determination')
  then('creates deliver stones: diagnosis, coverpage')
  then('binds route to current branch')
```

### permit.search (shell skill)

```
given('permit.search contract')
  then('exposes: permit.search --address <addr> --postal <code> [--limit N] [--since DATE] [--until DATE]')
  then('returns: permits array with number, type, status, date, contractor')
  then('returns: page metadata with limit, range, next cursor')
  then('returns: source jurisdiction and portal')
  then('throws: address not found error with format hint')
  then('throws: portal unavailable error with retry hint')
```

### permit.fetch (shell skill)

```
given('permit.fetch contract')
  then('exposes: permit.fetch --id <permit-id>')
  then('returns: permit number, type, status')
  then('returns: address')
  then('returns: work description')
  then('returns: contractor name or null')
  then('returns: inspections array with date and result')
  then('throws: permit not found error with format hint')
```
